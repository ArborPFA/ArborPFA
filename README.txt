INSTALL:
=========

Needs ilcsoft installed (http://ilcsoft.desy.de/portal). Tested with the following version :
 * v01-17-03

To install the package : 

  $ mkdir build
  $ cd build
  $ cmake -C $ILCSOFT/ILCSoft.cmake [-DPANDORA_MONITORING=ON] ..
  $ make
