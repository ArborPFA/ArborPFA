Download

The main package ArborPFA downloads and installs all sub packages for you. You can also download and install them one by one.

Use git command line tool to download it : 

git clone https://github.com/ArborPFA/ArborPFA.git
cd ArborPFA

You can choose your version using 'git tag' and set it 

 git tag
 ... list of tags ... 
git checkout  yourtag

Install

The algorithms themselves don't need any dependency. If you need to run in the ILC framework, then you will need ilcsoft installed in order to compile the MarlinArbor package. Warning : note that the current version of MarlinArbor uses the Gear geometry interface to construct the PandoraSDK geometry. A DD4HEP interface is ongoing ...

The build procedure is managed by CMake

Dependencies

All the needed packages are installed on-the-fly by CMake, except if you need to compile the MarlinArbor package.

In this case, you need the following packages installed within your ILCSOFT install :

  • Marlin (version > 1.0)
  • MarlinUtil

Note that the ROOT 3D event display monitoring can be built if ROOT is installed on your machine.

Installation procedure

In the root directory :

mkdir build
cd build
cmake [-DOPTIONS=...]  [-C /path/to/ILCSOFT.cmake] .. 
make

where OPTIONS can be :

  • PANDORA_MONITORING [ON/OFF] to install ROOT 3D Eve monitoring
  • MARLIN_ARBOR [ON/OFF] to install MarlinArbor package
  • INSTALL_DOC [ON/OFF] to install doxygen code documentation

Example :

cmake -DPANDORA_MONITORING=ON -DINSTALL_DOC=OFF ..

For ILCSOFT setup :

cmake -DPANDORA_MONITORING=ON -DINSTALL_DOC=OFF -C ../ILCSOFT.cmake ..
